# Personal Portfolio - AI Engineer

A single-page, high-performance portfolio website built with React, Vite, and Tailwind CSS. The design embodies "Controlled Futurism" with dynamic interactions and clean UI architecture, aimed to present AI Engineering capabilities effectively to recruiters and freelance clients.

## Tech Stack

- **Framework**: React 18, Vite
- **Styling**: Tailwind CSS v4, Custom Theming (Light/Dark mode)
- **Animation**: Framer Motion
- **UI Primitives**: Radix UI

## Getting Started

1. Clone the repository.
2. Ensure you have Node.js installed.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `src/app/` - Core application layout and section components
- `src/styles/` - Global styles and custom Tailwind theme configurations
- `src/imports/` - Assets and static PRD documentation

## Sections Overview

- **Hero**: Immediate identity and background animation layer.
- **About**: Bio and technical focus areas.
- **Projects**: Grid display of impactful AI systems implemented.
- **Skills**: Categorized technical proficiencies.
- **Insights**: Technical blog and thoughts on AI infrastructure.
- **Contact**: Reach-out calls to action and links.
